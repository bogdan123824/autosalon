# 🚗 AutoSalon

Сучасне веб-приложення для автосалону з повною функціональністю управління каталогом автомобілів, корзиною покупок та адміністративною панеллю.

## 📋 Опис

AutoSalon - це React-додаток, який дозволяє користувачам:
- Переглядати каталог автомобілів з детальною інформацією
- Додавати машини до кошика
- Оформляти замовлення тест-драйву
- Переглядати ціни та порівнювати характеристики
- Зв'язатися з салоном

Адміністратори мають доступ до панелі управління для редагування каталогу.

## 🏗️ Архітектура

- **Frontend**: React 18 + TypeScript
- **Стилізація**: CSS модулі
- **Маршрутизація**: React Router v6
- **Управління станом**: React Context API
- **База даних**: SQLite
- **Хуки**: Кастомні React хуки для логіки

## 🚀 Встановлення

### Вимоги
- Node.js 14+ 
- npm або yarn

### Кроки встановлення

1. **Клонування репозиторію**
```bash
git clone <ваш-репозиторій>
cd autosalon
```

2. **Встановлення залежностей**
```bash
npm install
```

3. **Запуск у режимі розробки**
```bash
npm start
```

Додаток буде доступний за адресою [http://localhost:3000](http://localhost:3000)

## 🗄️ База даних

Додаток використовує SQLite для зберігання даних. Таблиці:

- **cars** - Каталог автомобілів
- **gallery** - Галерея зображень
- **orders** - Замовлення
- **team** - Команда салону
- **connections** - Контактна інформація
